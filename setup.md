# This kit is now a Claude plugin

The RiftLab Personal AI OS is no longer installed by running this file. It installs through Claude's plugin system, plus a short interview.

If you were told to "follow setup.md," do these two steps instead:

1. **Install the plugin.** See [README.md](README.md) for the marketplace steps (Claude Desktop / Cowork and Claude Code). This adds the skills and agents.
2. **Run the interview.** Follow [INTERVIEW.md](INTERVIEW.md) to write your identity files (CLAUDE.md, voice, rules) into a folder you own.

Why the change: the plugin system installs and updates the skills and agents for you, so the old copy-everything recipe is no longer needed.
