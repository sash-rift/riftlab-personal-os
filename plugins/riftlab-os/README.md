# riftlab-os

The RiftLab Personal AI OS, as a Claude plugin.

Installs seven skills and the twelve agents that power them:

- `/aim-coach`, `/daily-brief`, `/meeting-prep`, `/humanize` (run anywhere, including Cowork)
- `/deep-research`, `/decision-council` (run a live agent team; use the Code tab or CLI)
- `/setup-my-context` (run once after installing to write your identity files)

After installing, run `/setup-my-context` (or tell Claude "help me set up my context") to write your CLAUDE.md, voice, and rules into a folder you own.

Built for the Applied AI for Knowledge Work course at RiftLab.
