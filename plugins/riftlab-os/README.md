# riftlab-os

The RiftLab Personal AI OS, as a Claude plugin.

Installs six skills and the eleven agents that power them:

- `/aim-coach`, `/daily-brief`, `/meeting-prep`, `/humanize` (run anywhere, including Cowork)
- `/deep-research`, `/decision-council` (run a live agent team; use the Code tab or CLI)

After installing, run the identity interview to personalize your setup. See [INTERVIEW.md](https://github.com/sash-rift/riftlab-personal-os/blob/main/INTERVIEW.md) in the repo, or tell Claude "run the RiftLab OS interview." It writes your CLAUDE.md, voice, and rules into a folder you own.

Built for the Applied AI for Knowledge Work course at RiftLab.
